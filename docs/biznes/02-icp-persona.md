## 📋 ICP Card (Karta Idealnego Klienta)

### Profil (1 akapit)

**Enpsyneia Check In** jest skierowany do **dorosłych osób (18-45 lat)**, które:
- Odczuwają chroniczne przeciążenie informacyjne i bodźcowe
- **Nawykowo sięgają po telefon bez konkretnego celu** (scrollowanie, sprawdzanie social media)
- Chcą ograniczyć korzystanie z social media, ale nie wiedzą czym je zastąpić
- Głównym job-to-be-done jest **szybkie rozpoznanie własnego stanu** w momencie dezorientacji i otrzymanie konkretnej, natychmiastowej podpowiedzi co zrobić

Użytkownik chce mieć **zdrowszą alternatywę** dla nawykowego sięgania po telefon.

### Persona Bullets

| Element | Opis |
|---------|------|
| **Branża** | Praca biurowa / praca zdalna / studia / sektor usługowy |
| **Rola** | Specjalista, menedżer, freelancer, student |
| **Wielkość firmy** | Dowolna (praca zdalna/stacjonarna) |
| **Wiek** | 18-45 lat |
| **Tryb pracy** | Praca przy komputerze, wiele meetingów, ciągła dostępność online |
| **Problem główny** | Nie wiem, czego teraz potrzebuję - ruchu, odpocznienia, ciszy? |
| **Problem nawykowy** | Nawykowo sięgam po telefon bez powodu, chcę to zmienić |
| **Trigger** | Moment przebodźcowania: po meetingach, wieczorem, przed podjęciem decyzji |
| **Trigger nawykowy** | "Znów scrollowałem 30 min" - szukam alternatywy |
| **Decision Criteria** | Szybkość (<2 min), prostota (minimalny friction), praktyczność (akcja, nie raport) |

| **KPI sukcesu** | Codziennie 1-3 check-iny, widoczna poprawa świadomości własnych potrzeb |

| **Budżet decyzyjny** | $0-15/miesiąc (jeśli freemium) |
| **Wartość marki** | Gotowość do polecenia aplikacji znajomym |

## 🔍 Job To Be Done (Job Snapshots)

### Job Snapshot: "Nawykowe sięganie po telefon"

| Element | Opis |
|---------|------|
| **Context** | Użytkownik siedzi bez celu, sięga po telefon "żeby coś zrobić" |
| **Motivation** | Chce przestać bezmyślnie scrollować, ale nie ma alternatywy |
| **Desired Outcome** | Aplikacja, którą odpala zamiast Instagram/TikTok - daje wartość w 30 sekund |
| **Current Solution** | Instagram → 30 min scrollowania → poczucie winy |
| **Barriers** | Aplikacja musi być szybka i dawać natychmiastową wartość |
| **Trigger** | Nuda, pustka, "coś zrobię na telefonie" |
| **Wartość** | Odzyskane 30-60 min dziennie, lepsze samopoczucie |
| **Confidence** | 8/10 |
| **Why this matters** | To jest UNIKALNA propozycja wartości - konkurencja nie oferuje tego |

### Job Snapshot 1: "Po maratonie spotkań" (bez zmian)

| Element | Opis |
|---------|------|
| **Context** | Użytkownik właśnie skończył 4-godzinny dzień spotkań online, czuje się "wyprany" |
| **Motivation** | Chce wiedzieć, czy potrzebuje przerwy, czy jeszcze może coś zrobić |
| **Desired Outcome** | Jedno konkretne działanie: "idź na spacer" lub "zrób 5 minutstretchingu" w mniej niż 1 minutę |
| **Current Solution** | Scroluje Instagram, pije kolejną kawę, albo "po prostu się zmusza" |
| **Barriers** | Nie ma czasu na długie aplikacje, nie chce "journalingu", potrzebuje <1 min |
| **Trigger** | Koniec serii spotkań, godzina 15-17 |
| **Wartość** | 15-30 min odzyskanego czasu, uniknięcie wypalenia |
| **Confidence** | 7/10 |

## 📝 Interview Script (nowe pytania o nawyki)

### Pytania

1. **"Opowiedz mi o ostatnim razie, kiedy nie wiedziałeś, co ze sobą zrobić po pracy."**
   - Kontekst: gdzie byłeś, co robiłeś, jak się czułeś?

2. **"Co dokładnie robiłeś krok po kroku?"**
   - Current solution: jak rozwiązałeś problem (lub nie)?

3. **"Co w tym procesie jest najbardziej frustrujące?"**
   - Pain: co Ci przeszkadza najbardziej?

4. **"Jak rozpoznasz, że problem został rozwiązany?"**
   - Desired outcome: jaki jest Twój ideał?

5. **"Ile czasu/kosztu to zabiera teraz?"**
   - Value: ile minut/godzin tracisz?

6. **"Co byś zapłacił za prostsze rozwiązanie?"**
   - Willingness-to-pay: nawet $0 jest informacją

7. **"Czy próbowałeś już czegoś innego? Dlaczego to nie zadziałało?"**
   - Alternatives: co już testowałeś?

8. **"Opowiedz mi o swoim nawyku korzystania z telefonu."**
   - Jak często sprawdzasz telefon bez konkretnego powodu?
   - Kiedy najczęściej to robisz?
   - Co czujesz po godzinie scrollowania?

9. **"Gdyby istniała aplikacja, która mogłaby zastąpić Twoje nawykowe sięganie po telefon, co byś chciał/chciała żeby robiła?"**
   - Czas: ile minut dziennie by zaoszczędziła?
   - Funkcja: co konkretnie by oferowała?
