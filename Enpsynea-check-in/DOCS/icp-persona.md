## 📋 ICP Card (Karta Idealnego Klienta)

### Profil (1 akapit)

**Enpsyneia Check In** jest skierowany do **dorosłych osób (25-45 lat) pracujących zawodowo**, które odczuwają chroniczne przeciążenie informacyjne i bodźcowe. Głównym job-to-be-done jest **szybkie rozpoznanie własnego stanu w momencie dezorientacji i otrzymanie konkretnej, natychmiastowej podpowiedzi co zrobić**. Największym bólem jest niemożność podjęcia decyzji o tym, czego się potrzebuje w danym momencie - ruchu, odpoczynku, ciszy czy kontaktu z drugą osobą.

---

### Persona Bullets

| Element | Opis |
|---------|------|
| **Branża** | Praca biurowa / praca zdalna / sektor usługowy |
| **Rola** | Specjalista, menedżer, freelancer |
| **Wielkość firmy** | Średnie i duże przedsiębiorstwa (50+ pracowników) |
| **Wiek** | 25-45 lat |
| **Tryb pracy** | Praca przy komputerze, wiele meetingów, ciągła dostępność online |
| **Problem główny** | Nie wiem, czego teraz potrzebuję - ruchu, odpoczynku, ciszy? |
| **Trigger** | Moment przebodźcowania: po meetingach, wieczorem, przed podjęciem decyzji |
| **Decision Criteria** | Szybkość (<2 min), prostota (bez logowania), praktyczność (akcja, nie raport) |
| **KPI sukcesu** | Codziennie 1-3 check-iny, widoczna poprawa świadomości własnych potrzeb |
| **Budżet decyzyjny** | $0-15/miesiąc (jeśli freemium) |

## 🔍 Job To Be Done (Job Snapshots)

Poniżej przedstawiam **5 typowych Job Snapshots** - sytuacji, w których użytkownik sięga po rozwiązanie:

### Job Snapshot 1: "Po maratonie spotkań"

| Element | Opis |
|---------|------|
| **Context** | Użytkownik właśnie skończył 4-godzinny dzień spotkań online, czuje się "wyprany" |
| **Motivation** | Chce wiedzieć, czy potrzebuje przerwy, czy jeszcze może coś zrobić |
| **Desired Outcome** | Jedno konkretne działanie: "idź na spacer" lub "zrób 5 minutstretchingu" |
| **Current Solution** | Scroluje Instagram, pije kolejną kawę, albo "po prostu się zmusza" |
| **Barriers** | Nie ma czasu na długie aplikacje, nie chce "journalingu", potrzebuje <1 min |
| **Trigger** | Koniec serii spotkań, godzina 15-17 |
| **Wartość** | 15-30 min odzyskanego czasu, uniknięcie wypalenia |
| **Confidence** | 7/10 |

---

### Job Snapshot 2: "Wieczorna dezorientacja"

| Element | Opis |
|---------|------|
| **Context** | Wieczór po pracy, użytkownik siedzi przed TV/ekranem, nie może zasnąć |
| **Motivation** | Chce się zrelaksować ale nie wie, co by mu pomogło |
| **Desired Outcome** | Jasna odpowiedź: "teraz potrzebujesz ciszy" lub "potrzebujesz lekkiego ruchu" |
| **Current Solution** | Ogląda Netflix, scrolluje social media, "jakoś minie" |
| **Barriers** | Aplikacje wellness są za długie, nudne, wymagają zaangażowania |
| **Trigger** | 21-23 wieczorem, przed snem |
| **Wartość** | Lepszy sen, 30-60 min mniej straconego czasu |
| **Confidence** | 6/10 |

---

### Job Snapshot 3: "Poranna niepewność"

| Element | Opis |
|---------|------|
| **Context** | Ranek przed pracą, użytkownik nie może się zebrać, "coś go trzyma" |
| **Motivation** | Chce szybko ustawić ton dnia, wiedzieć czego unikać |
| **Desired Outcome** | Krótki raport: "dziś twój dzień działania" lub "dziś potrzebujesz spokoju" |
| **Current Solution** | Sprawdza telefon, pije kawę bezmyślnie, "jakoś będzie" |
| **Barriers** | Nie ma czasu na długi poranny rytuał, potrzebuje <2 min |
| **Trigger** | 6-8 rano, przed wyjściem do pracy |
| **Wartość** | Produktywność +10-20%, lepsze planowanie dnia |
| **Confidence** | 5/10 |

---

### Job Snapshot 4: "Kryzys w pracy"

| Element | Opis |
|---------|------|
| **Context** | Środek dnia pracy, atak paniki/lęku, potrzeba "złapania oddechu" |
| **Motivation** | Chce szybko się uspokoić bez wychodzenia z biura |
| **Desired Outcome** | Prosta technika: "weź 3 głębokie oddechy" lub "wyjdź na 2 min" |
| **Current Solution** | Idzie do łazienki, pije wodę, próbuje "przeczekać" |
| **Barriers** | Nie może wyjść z budynku, potrzebuje dyskretnego rozwiązania |
| **Trigger** | 10-14 w pracy, w trakcie stressogennego zadania |
| **Wartość** | Uniknięcie kryzysu, kontynuacja pracy |
| **Confidence** | 8/10 |

---

### Job Snapshot 5: "Nadmierna analiza"

| Element | Opis |
|---------|------|
| **Context** | Użytkownik "myśli za dużo", nie może podjąć prostej decyzji |
| **Motivation** | Chce "wyłączyć głowę" i pozwolić sobie na prostą akcję |
| **Desired Outcome** | Ktoś/d coś mu powie co ma zrobić, zamiast samemu decydować |
| **Current Solution** | Decyduje się na "nic" albo robi przypadkową rzecz |
| **Barriers** | Paraliż decyzyjny, potrzebuje zewnętrznego bodźca |
| **Trigger** | Kiedykolwiek, wiele razy dziennie |
| **Wartość** | Oszczędność 10-30 min, mniej stresu |
| **Confidence** | 7/10 |

## 📝 Interview Script (7 pytań do JTBD)

1. **Opowiedz o ostatnim razie, kiedy nie wiedziałeś, co ze sobą zrobić po pracy.**
   - Kontekst: gdzie byłeś, co robiłeś, jak się czułeś?

2. **Co dokładnie robiłeś krok po kroku?**
   - Current solution: jak rozwiązałeś problem (lub nie)?

3. **Co w tym procesie jest najbardziej frustrujące?**
   - Pain: co Ci przeszkadza najbardziej?

4. **Jak rozpoznasz, że problem został rozwiązany?**
   - Desired outcome: jaki jest Twój ideał?

5. **Ile czasu/kosztu to zabiera teraz?**
   - Value: ile minut/godzin tracisz?

6. **Co byś zapłacił za prostsze rozwiązanie?**
   - Willingness-to-pay: nawet $0 jest informacją

7. **Czy próbowałeś już czegoś innego? Dlaczego to nie zadziałało?**
   - Alternatives: co już testowałeś?

## ⚠️ Ostrzeżenie (z Kill-The-Idea)

**Ten ICP ma fundamentalny problem biznesowy:**

- Rynek wellness = darmowy (ludzie nie płacą za apki wellness)
- localStorage = brak kont = brak monetyzacji
- "Przebodźcowani" = trudna grupa do dotarcia (CAC wysoki)

**Rekomendacja:** Po walidacji ICP rozważ pivota w kierunku:
- Węższa nisza (studenci w sesji, osoby z ADHD)
- Model freemium z kontami (Supabase/Firebase)
- B2B (employee wellness jako benefit pracowniczy)
