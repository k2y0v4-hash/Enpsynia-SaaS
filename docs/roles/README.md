# SDD Roles

Formalna dokumentacja ról Spec Driven Development znajduje się w tym katalogu.

Wcześniejszy praktyczny system ról agentów pozostaje w:

- `agents/ROUTING.md`
- `agents/roles/`

Te dwa systemy pełnią różne funkcje:

- `docs/roles/` opisuje role jako część dokumentacji SDD repozytorium.
- `agents/roles/` opisuje praktyczny routing pracy agentów AI.

Nie usuwamy `agents/roles`. Przy sprzeczności między rolami a aktualnym planem funkcjonalności obowiązuje zatwierdzony plan `docs/plans/PLAN_*.md` oraz decyzja właściciela.
